# 社团管理平台

这是一个基于 `Spring Boot + Vue 3` 的社团管理平台重构项目，当前已经完成可演示的 MVP 版本，并具备继续扩展的工程基础。

## 当前目录

- `backend`：后端工程
- `frontend/app`：前端工程
- `db`：数据库初始化脚本与补丁
- `docs`：项目文档

## 当前已实现功能

- 用户注册、登录与 JWT 鉴权
- 游客公开浏览社团、活动、公告
- 社团列表查看与入社申请
- 入社申请审批与撤销
- 活动发布、报名、取消报名、报名名单查看
- 公告中心
- 成员管理
- 个人中心
- 系统管理员用户管理
