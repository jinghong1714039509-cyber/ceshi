# 社团管理平台重构计划

## 工作区说明

- `backend`：当前正在维护的 Spring Boot 后端
- `frontend`：新的 Vue 3 + Vite 前端
- `db`：数据库初始化脚本和增量补丁
- `docs`：重构说明、启动手册、演示手册
- `original`：旧项目归档，仅作参考，不再继续开发

## 已完成阶段

### 第一阶段：工程基线重构

- 整理旧项目目录，保留唯一后端基线
- 后端升级到 `Spring Boot 2.7.18 + Java 17`
- 引入基础工程配置：OpenAPI、统一异常处理、CORS、缓存配置
- 拆出数据库初始化脚本和首个安全补丁

### 第二阶段：认证鉴权改造

- 新增 `JWT + Spring Security`
- 新增 `/api/auth/login` 与 `/api/auth/me`
- 明文密码兼容迁移到 `BCrypt`
- 旧接口可以通过 Bearer Token 兼容过渡

### 第三阶段：前端重建

- 新建 `Vue 3 + Vite + Element Plus` 前端
- 完成登录页、首页、社团列表、申请审批、活动管理、公告中心、成员管理页面

## 当前已完成的业务闭环

### 闭环一：入社申请

- 学生浏览社团
- 学生提交入社申请
- 管理员查看申请列表
- 管理员审批通过
- 成员关系自动写入 `members`

### 闭环二：活动报名

- 管理员发布活动
- 学生查看活动并报名
- 管理员查看报名名单

## 当前系统角色

- 学生：浏览社团、申请入社、报名活动、查看公告
- 社团管理员：审批申请、发布活动、查看报名名单、发布公告、管理成员
- 系统管理员：具备更高权限，可查看全局数据并发布系统公告
